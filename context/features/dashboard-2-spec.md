# Dashboard UI Phase 2 Spec

## Overview

This is phase 2 of 2 for the dashboard UI layout. Use the screenshot referenced below for how it should look. Use the dummy data to fill data fields.

## Requirements for phase 2

- Collapsible sidebar
- Option to go PREV/NEXT slide
- Possibility to open/close sidebar clicking table of contents button
- Drawer icon to open/close sidebar
- Always a drawer on mobile view
- Add example markdowns
